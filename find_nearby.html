Nearby discovery

Find SugarBabies or SugarDaddies near you.

Tap one of the discovery buttons below. We will ask for your location permission, then log your selected search type and approved location so you can build a nearby matching flow around it.

This page logs page visits through your Vercel middleware. When a visitor presses a button, it also logs their IP address again together with the chosen action and any location they allow the browser to share.

What gets logged

Page visit Each visit to this page is logged with IP address, path, user agent, referer, and timestamp.
Discovery action The button choice is logged as either `find_sugarbabies_near_me` or `find_sugardaddies_near_me`.
Location result If permission is granted, latitude, longitude, and accuracy are logged. If permission is denied, that denial is logged too.
